HOW TO JOIN

Java Edition (PC)

  1. Open Minecraft Java Edition.
  2. Click Multiplayer, then click Add Server.
  3. In the Server Name field, type: Zefty MC.
  4. In the Server Address field, enter:
    play.zeftymc.net
  5. Click Done, select Zefty MC from the list, then click Join Server.

Bedrock Edition (Mobile / Console / Win10)

  1. Open Minecraft Bedrock Edition.
  2. Go to the Play menu, then open the Servers tab.
  3. Scroll down and click Add Server.
  4. Server Name: Zefty MC
  5. Server Address: bedrock.zeftymc.net (or your IP)
    Port: 19132 (default Bedrock port)
  6. Click Save, then tap the server and hit Join.

Need Help?

If you have trouble joining, make sure your Minecraft is on the correct version and your internet connection is stable. You can also join our Discord for support and live help from staff and players.
